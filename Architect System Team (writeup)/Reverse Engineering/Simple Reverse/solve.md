kita download dulu soal yang tadi dan pergi ke terminal Linux nya

kita mencoba untuk memberikan permission pada file tersebut dengan
```plaintext
$chmod +x let
```
dan kita mencoba untuk menjalankan file tersebut dengan perintah 
```plaintext
$./let
```
